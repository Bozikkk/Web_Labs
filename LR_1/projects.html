Цікаві радіотехнічні проєкти

Радіоелектроніка дозволяє створювати безліч корисних пристроїв — від найпростіших підсилювачів до складних систем комунікації. Нижче наведені приклади популярних проєктів для початківців:

-Металошукач
Пристрій для виявлення металевих предметів на основі принципу електромагнітної індукції.
-FM-трансмітер
Пристрій для передачі сигналів на FM-частотах для прослуховування через стандартний радіоприймач.
-FM-радіоприймач
Пристрій для прийому та, можливо, обробки сигналів на FM-частотах.
-Бездротовий зарядний пристрій
Система для передачі електроенергії без використання дротів на основі електромагнітної індукції.
